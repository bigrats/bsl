示例代码

<!--[
    // 一参数include，参数为常量，直接指定了模板，可以是相对路径，
    // 使用当前include位置在上下文信息构建被引入模板的上下文，
    // 被引入的模板可以使用include位置上可以使用的任何变量
    include("../xxx.html");

    // 一参数include，参数为表达式，结果为要引入的模板路径，可以是相对路径，
    // 使用当前include位置在上下文信息构建被引入模板的上下文，
    // 被引入的模板可以使用include位置上可以使用的任何变量
    include(expression);

    // 两参数include，参数为表达式，结果为要引入的模板路径，可以是相对路径，参数二是附加的HashMap格式的参数
    // 被引入的模板可以使用当前include位置在上下文中可以使用的任何变量与HashMap的组合，
    // 后者覆盖前者后构建为被引入模板的上下文
    include(expression, {'a':'value'});
]-->

代码解释：

    1、引入模板时可以使用一参数或二参数的include指令，被引入模板的上下文由当前模板的上下文及附加的HashMap联合构建；
    2、使用引入模板可以完成复杂的布局，即将需要制作的布局切割成小的模板文件，通过include进行整合；
    3、支持include嵌套，即include的模板里是可以继续include的；
    4、利用强大的include指令可以构建出许多富有创造性的组件，例如我们可以构建Web组件库；
