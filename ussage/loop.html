示例代码

<!--[
    // 循环items集合，将集合中的每个对象交给定义的item变量
    // 在此items可以是集合类型（Array，List，Map），也可以是基本类型（数字、布尔值）
    loop(item : items) {
        // 输出
        echo(item.price);
    }
]-->

代码解释：

    1、在Bsl中只有loop循环，语法非常简单；
    2、在Bsl中可以循环集合类型（Array，List，Map），也可以循环基本类型（数字、布尔值）；
    3、不管是上述的集合类型还是基本类型在交由Bsl执行时都会被封装为Iterator类型进行迭代；
    4、循环迭代中的{ }作用域块内可以定义属于该作用域的变量；
