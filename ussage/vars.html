示例代码

<!--[
    // 定义字节
    var a=10b, b=10B;

    // 定义短整型
    var c=10s, d=10S;

    // 定义整数
    var e=10;

    // 定义长整型
    var f=10l, g=10L;

    // 定义单精度浮点数
    var h=10.5f, i=10.5F;

    // 定义双精度浮点数
    var j=10.5d, k=10.5D;

    // 定义字符串
    var l='test1', m="test2";

    // 定义变长数组
    var n=['test1', 'test2', l, m];

    // 定义哈希表
    var o={'k1': 'v1', "k2": "v2", l: m};

    // 复合定义
    var p=a+b*10, q='value_'+c+d, r={'k1': 'v1', "k2": "v2", l: m, 'value': q};

    // 不同作用域下的变量定义，可以定义同名的，但不能与外层作用域的变量冲突
    if(a==10) {
        var s=100;
    } else {
        var s=101;
        // 下面的定义是错误的，因为变量a已在外层定义
        var a=100;
    }
]-->

代码解释：

    1、本引擎使用类似JavaScript的弱类型定义方式，也可以一次性定义多个；
    2、支持类似JSON串的定义方式，参考变长数组及哈希表定义；
    3、支持使用已定义的变量进行运算后赋值定义；
    4、支持不同作用域下的变量定义，{ }各自持有自己的作用域；
       外层变量可以被内层变量使用，内层变量不可以被外层使用，与Java的变量作用域类似；
       例如循环和多路分支都有作用域块；
    5、任何类型都可以进行逻辑运算（会转换成布尔类型，默认约定是布尔的true、false，数字非0即真，对象非空即真，参考多路分支 ）；
