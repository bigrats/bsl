    if(a > 30) {
        // 语句 ...
        echo(item.change);

注意事项：
    1、多路分支跟Java和JavaScript中的非常相似，但是多路分支中的条件可以是表达式，任意对象或基本类型数据；
    2、多路分支中的{ }作用域块内可以定义属于该作用域的变量；
