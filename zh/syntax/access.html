示例代码：
<!--[
    // 静态属性、常量、提供Getter方法的非public属性都可以使用'.'操作符访问
    var a=item.price;

    // 直接调用方法设置属性值
    item.setPrice(10);

    // 调用对象的方法，无参数的
    var b=item.getPrice();

    // 调用对象的方法，带参数的
    var c=item.doXxx(param1, param2);

    // 连续的属性方法调用
    var d=item.field1.doXxx(param1, param2).doYyy(param);

    // List内的取值
    var e=list.get(0);
    var f=list.get(整数);

    // Map内的取值
    var g=map.get("key");
    var g=map.get(对象);
]-->
注意事项：
    1、使用'.'操作符可以访问对象所有的public属性及提供了Getter方法的非public属性；
    2、支持直接调用对象的Getter方法获取属性值，直接调用对象的Setter方法设置属性值；
    3、支持直接调用对象的无参数或有参数方法；
    4、支持连续的属性方法调用；
    5、属性及方法调用属于表达式运算；
